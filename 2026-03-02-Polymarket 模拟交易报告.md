## 🎯 七、实盘启用步骤

### 7.1 必需配置

```bash
# 1. 编辑配置文件
nano /root/.openclaw/workspace-dev/polymarket-bot/config.env

# 2. 填写以下必填项：
PRIVATE_KEY=你的钱包私钥
FUNDER_ADDRESS=你的资金地址
POLYGON_RPC_URL=https://polygon-rpc.com  (或 Alchemy/Infura)

# 3. 开启自动交易
AUTO_TRADE=true

# 4. 保存并重启
cd /root/.openclaw/workspace-dev/polymarket-bot
python3 polymarket_auto_trade.py
```

### 7.2 资金准备

| 项目 | 金额 | 说明 |

| **USDC** | ≥1000 USDC | 交易资金（建议 10 单） |
| **MATIC/POL** | ≥10 MATIC | Gas 费用 |
| **总计** | ≈$1010 | 启动资金 |
